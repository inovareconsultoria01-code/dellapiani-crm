# Dellapiani CRM

Sistema simples inspirado no Kommo para gestão de clientes, funil comercial e WhatsApp.

## Login
Usuário: `admin`  
Senha: `1234`

## Funções
- Cadastro de clientes
- Funil de vendas
- Dashboard
- Ranking por vendedor
- Botão para chamar cliente no WhatsApp
- Dados salvos no navegador via LocalStorage

## Como usar
1. Abra o arquivo `index.html` no navegador.
2. Para publicar, envie os arquivos para GitHub, Netlify ou Vercel.
3. Depois podemos integrar com Firebase, Supabase ou WhatsApp API.
