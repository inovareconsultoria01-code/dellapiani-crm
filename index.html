Na versão online com servidor, cada número é vinculado por QR Code usando Evolution API ou WhatsApp Cloud API.

Como vincular todos os celulares

1. Cada vendedor terá um número WhatsApp Business.

2. O gestor adiciona o número na aba Celulares.

3. Na versão com servidor, o sistema gera QR Code para cada número.

4. Após escanear, todas as mensagens entram na inbox do gestor.

5. O gestor acompanha atendimento, tempo de resposta, leads e vendas.
